CODE PROJECTS

I spend most of my time coding professionally in modern JavaScript frameworks like React and React Native. In my free time coding I try out different languages, explore and customize my environments and tooling, and use my coding and data science skills in areas that I'm especially passionate about, like politics, the environment and music creation.

Below, I will briefly highlight some projects, with links to more info where appropriate.


Project: Portfolio (This Site!)

The idea behind this portfolio site is to showcase a little bit about who I am and where my intellectual and creative energy is spent.

I have chosen to build this site from scratch using only vanilla web languages, wihtout the assistance of any third-party libraries, frameworks or templates. It feels like it has been so long since I have actually worked with vanilla HTML/CSS/JS, and I felt these foundations of the web get a bad rap given how hyped frameworks are. But for a simple portfolio site, I don't need a Virtual DOM, server-side rendering, or other bells and whistles that pay dividends only at scale.

That said, I may at some point refactor this site in a framework like React, just for fun and to explore some less-than-production-ready features and plugins.


Project: Marxian Split

This is a fun little project to split expenses between 2 parties based on relative income.

I don't know how useful it is, but it's a way to think about income inequality and put your money where your mouth is. Right now it's just an in-browser script, but I intend to use it as a 'Hello World' app for trying out different frameworks with a little more logic than a print command.
